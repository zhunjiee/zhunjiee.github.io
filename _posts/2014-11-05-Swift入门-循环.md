##for循环

* 基本用法和OC一致

* 条件表达式必须是bool类型的值

* 条件表达式的()可以省略

* 在OC中如果{}中只有一条语句, 那么{}可以省略, 而Swift不可以



```

for var i = 0; i < 10; i++

{

    print(i)

}

```



##区间

**半闭区间**: 0..<10  包含头不包含尾

**闭区间**:   0...10  包含头包含尾



```

for i in 0..<10

{

    print(i)

}

```



##while循环

* 基本用法和OC一致

* 条件表达式必须是bool类型的值

* 条件表达式的()可以省略

* 在OC中如果{}中只有一条语句, 那么{}可以省略, 而Swift不可以



```

var num = 0

while num < 10

{

    print(num)

    num++

}



/*:

repeat while循环

* 基本用法和OC一致

* 条件表达式必须是bool类型的值

* 条件表达式的()可以省略

* 在OC中如果{}中只有一条语句, 那么{}可以省略, 而Swift不可以

*/

var value = 0

repeat{

    print(value)

    value++

}while value < 10

```
